# Exercice : Fonction de validation avec génériques

## Objectif

Créer une fonction générique `ensureEverythingIsDefined` qui prend un objet avec des propriétés optionnelles (ou non) et retourne un nouvel objet où toutes les propriétés deviennent obligatoires (non-undefined).

## Exemple

```typescript
interface Person {
    name?: string;
    age?: number;
}

const person: Person = { name: "Alice", age: undefined };

const validatedPerson = ensureEverythingIsDefined(person);

validatedPerson.name; // Type: string (et non string | undefined)
// Devrait lancer une erreur à l'exécution car age est undefined
```
