# Installation et outils requis pour ce cours

- npm (installé avec Node.js) : pour installer TypeScript et d'autres outils.
- Un éditeur de code comme Visual Studio Code (VSCode) : pour écrire et gérer votre code TypeScript.
- Le compilateur TypeScript (tsc) : pour compiler le code TypeScript en JavaScript.

La version de NodeJS doit être >= 24.x
La version de TypeScript utilisée dans ce cours est la 5.9.x
